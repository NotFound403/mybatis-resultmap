## 使用方式

使用了 h2 数据库内存模式  启动后`ddl.sql`和`dml.sql` 会初始化数据库。 

- 启动 springboot 项目  然后点击 [H2 数据库控制台](http://localhost:8080/h2-console)。

- 登录用户 `sa`  密码不填  直接点击 **connect** 按钮进入，选择 schema `mybatis` 。
